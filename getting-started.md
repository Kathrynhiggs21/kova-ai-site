# Getting Started

Welcome to KOVA's public docs.

- App code: private `kova-ai` repo (Netlify/Vercel deploys)
- This site: `kova-ai-site` (GitHub Pages)

## Quickstart
1. Keep your app code in the private repo.
2. Edit this site (`index.html` and `/docs/*`) for public-facing info.
3. Commit to `main` to publish via GitHub Pages.
