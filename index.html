What is KOVA?

KOVA stands for Katy's Organizational Voice Assistant. It orchestrates your notifications, schedules, files, and research—then ships clean summaries to your email and phone.

How it's built

Private app repo (kova-ai) deploys to Netlify/Vercel. Public repo (kova-ai-site) powers this GitHub Pages site.

Privacy first

Your production code and secrets live in the private repo. Public docs only—no sensitive data here.
